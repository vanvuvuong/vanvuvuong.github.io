# Sử Dụng Cửa Sổ Câu Lệnh (Terminal) Linux Nhanh Gấp 10 Lần Người Bình Thường


Bài này sẽ hướng dẫn bạn sử dụng của sổ câu lệnh một cách "khoa học"

## Làm bạn với phím tắt

### Di chuyển con trỏ

| Mô tả                      | Phím tắt                        |
| -------------------------- | ------------------------------- |
| Nhảy về phía đầu/cuối dòng | `ctrl+a/e`                      |
| Tiến/lùi bề một ký tự      | `ctrl+f/b`                      |
| Tiến/lùi bề một từ         | `alt+f/b` hoặc `ctrl+phải/trái` |

### Điều khỉển kết quả hiển thị

| Mô tả                                                                                                   | Phím tắt |
| ------------------------------------------------------------------------------------------------------- | -------- |
| Xóa toàn bộ kết quả ở màn hình                                                                          | `ctrl+l` |
| Tạm dừng cập nhật kết quả ở màn hình<br>(kiểu như kết quả đang chạy liên tục, bấm phím tắt sẽ tạm dừng) | `ctrl+s` |
| Tiếp tục cập nhật kết quả ở màn hình (ngược với cái trên)                                               | `ctrl+q` |

### Chỉnh sửa

| Mô tả                         | Phím tắt         |
| ----------------------------- | ---------------- |
| Xóa 1 ký tự trước/sau con trỏ | `ctrl+h/d`       |
| Xóa 1 từ trước/sau con trỏ    | `ctrl+w`/`alt+d` |
| Xóa cho đến đầu/cuối dòng     | `ctrl+u/k`       |
| Dán kết quả vừa xóa           | `ctrl+y`         |
| Hoàn tác thao tác cuối cùng   | `ctl-+shift+-`   |

### Tra cứu lịch sử (câu lệnh)

| Mô tả                     | Phím tắt               | Ghi chú thêm                                                                                                                                                |
| ------------------------- | ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tìm kiếm trong lịch sử    | `ctrl+r`               | Sau khi bấm phím tắt, nhập vài ký tự liên quan tói câu lệnh bạn muốn tìm<br> rồi tiếp tục bấm `ctrl+r` để tìm ngược trong lịch sử, bấm `ctrl+s` để tìm xuôi |
| Câu lệnh trước đó/kế tiếp | `ctrl+p/n` / `up/down` | Bấm liên tục để tìm kiếm các câu lệnh sâu phía trước                                                                                                        |

### Tiến trình

| Mô tả                                                                           | Phím tắt |
| ------------------------------------------------------------------------------- | -------- |
| Chuyển tiến trình hiện tại xuống chạy ngầm (không hiển thị kết quả ra màn hình) | `ctrl+z` |
| Tắt tiền trình hiện tại                                                         | `ctrl+c` |
| Thoát khỏi cửa sổ câu lệnh                                                      | `ctrl+d` |
